# Ecological Networks in Human-Dominated Environments

**MSc Thesis | MSc Computer Science (Data Analytics) | University of Galway | 2023**

## Overview
This thesis investigates the fragility of ecological networks when key 
species are removed from an ecosystem. It examines how network structure 
changes under species loss and what thresholds cause a network to shift 
from stable to fragile.

## Research Questions
- How does removing a particular species affect the structure and 
  functioning of ecological networks?
- What levels of species extinction cause a network to become fragile?
- What can agent-based model simulations reveal about the effects 
  of species extinction?

## Methodology
- Network analysis of food web datasets (AkatoreA and Ythan92)
- Systematic removal of individual species (nodes) from food webs
- Agent-based modelling (ABM) to simulate percentage-based species removal
- Analysis of structural changes in network metrics

## Tools & Languages
- R (RMarkdown)
- Network analysis libraries
- Agent-based modelling frameworks

## Files
- `AkatoreA_Code.Rmd` — Analysis code for the AkatoreA food web dataset
- `Ythan_Code.Rmd` — Analysis code for the Ythan92 food web dataset

- `Ythan92.xlsx` — Ythan92 ecosystem dataset
